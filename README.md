# interview-qa

后端面试知识库，按主题整理 Go、数据库、中间件、基础知识、系统设计、算法、公司面经和 AI 工具链相关内容。

## 目录

- `go/`：Go 语言基础、并发、运行时等
- `database/`：MySQL、Redis
- `middleware/`：Kafka、gRPC、etcd、Docker、K8s、Elasticsearch
- `fundamentals/`：Linux、操作系统、计算机网络
- `system-design/`：系统设计、场景题、稳定性与故障隔离
- `algorithm/`：算法题整理
- `company/`：公司面经
- `ai-tools/`：AI 工具链相关
- `cross-topic/`：跨主题高频补充

## 协作方式

欢迎通过 Fork + Pull Request 提交内容。建议每个 PR 聚焦一个主题，保持目录分类清晰，避免重复题目。

## 内容说明

当前内容来自个人面试复习资料整理，后续可以继续补充 Java、Python、云原生、测试开发、项目面试等方向。
